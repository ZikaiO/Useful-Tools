# Construct String by String Builder
```java
    StringBuilder str_bid = new StringBuilder();
    str_bid.append(0);
    str_bid.append(1);
    str_bid.append(2);
    str_bid.append(3);
    str_bid.append(4);
    System.out.println(str_bid.toString());
    // 1. Delete Char By Range, include start index but not include the end index.
    str_bid.delete(0,2);
    System.out.println(str_bid.toString());
    // 2. Delete Char By index
    str_bid.deleteCharAt(str_bid.length() - 1);
    System.out.println(str_bid.toString());
```
